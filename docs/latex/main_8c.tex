Half-\/\+Second interrupt that controls our send message routine of the S\+SN. Half-\/second and not one second is because we can not set an interrupt of up to 1 second with the current clock of the S\+SN. We only start this interrupt service once we have Ethernet configured and all self-\/tests are successful. The message to be sent is constructed every half a second in the main function and only reported to the server after every \char`\"{}\+S\+S\+N\+\_\+\+R\+E\+P\+O\+R\+T\+\_\+\+I\+N\+T\+E\+R\+V\+A\+L\char`\"{} seconds. \mbox{\Hypertarget{main_8c_ae66f6b31b5ad750f1fe042a706a4e3d4}\label{main_8c_ae66f6b31b5ad750f1fe042a706a4e3d4}}

{\footnotesize\ttfamily int main (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}

The main loop of S\+SN operation. It calls the following functions in order\+:
\begin{DoxyItemize}
\item Sets up all required peripherals
\item Runs system tests for checking\+:
\begin{DoxyEnumerate}
\item E\+E\+P\+R\+OM Read/\+Write
\item Temperature and Humidity Sensor
\item Ethernet Physical Connection
\end{DoxyEnumerate}
\item Finds M\+AC address in E\+E\+P\+R\+OM; if available, assigns it to S\+SN; if not available, assigns default M\+AC address to S\+SN
\item Sets up Ethernet connection using which ever M\+AC address was selected
\item If using default M\+AC address, S\+SN sends periodic G\+E\+T\+\_\+\+M\+AC requests to S\+SN Server until it successfully retrieves one and resets self
\item Waits for five seconds for new Current Sensor Configurations from S\+SN Server. These cannot be reprogrammed after this five seconds window
\item If new configurations received, assigns them to S\+SN, writes them to E\+E\+P\+R\+OM and proceeds
\item If new configurations not received, finds Current Sensor Configurations in E\+E\+P\+R\+OM; if available, assigns them to S\+SN; if not available S\+SN sends periodic G\+E\+T\+\_\+\+C\+O\+N\+F\+IG requests to S\+SN Server until it successfully retrieves one and writes them to E\+E\+P\+R\+OM and proceeds
\item S\+SN sends periodic G\+E\+T\+\_\+\+Time\+Of\+Day requests to S\+SN Server until it successfully retrieves it and proceeds
\item S\+SN starts global clock and half-\/second periodic interrupt
\item S\+SN calculates machine status update and ambient conditions every 100 milliseconds. The I\+SR sends the status update after every \$\{S\+S\+N\+\_\+\+R\+E\+P\+O\+R\+T\+\_\+\+I\+N\+T\+E\+R\+V\+AL\} seconds 
\end{DoxyItemize}
